# EBV-OncoImmunoVax

## Overview
**EBV-OncoImmunoVax** is an immunoinformatics-driven repository supporting the rational design and evaluation of multi-epitope vaccine candidates targeting **oncogenic Epstein–Barr virus (EBV) proteins**. The pipeline integrates epitope prediction, antigenicity assessment, immune simulation, and structural refinement with a focus on EBV-associated malignancies.

This repository accompanies a research manuscript and is intended to promote **reproducibility, transparency, and downstream experimental translation**.

---

## Scientific Scope
- EBV oncogenic and latency-associated proteins
- MHC class I and II epitope mapping
- Proteasomal cleavage prediction (NetChop)
- B-cell and T-cell epitope prioritization
- Antigenicity and immunogenicity analysis
- Immune response simulation (C-ImmSim)
- 3D structure prediction and refinement of multi-epitope constructs
